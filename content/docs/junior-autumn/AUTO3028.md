## 教材

- **数学规划及其应用**（范玉妹 等 / 机械工业出版社 / 2018 / textbook）
- **运筹学**（教材编写组 / 清华大学出版社 / 2005 / textbook）
- **数学规划基础**（刘红英 等 / 北京航空航天大学出版社 / 2012 / textbook）
- **数据、模型与决策**（安得森 等 / 机械工业出版社 / 2012 / textbook）
- **运筹学的思想方法及应用**（焦宝聪 / 北京大学出版社 / 2008 / textbook）
- **运筹学与最优化方法**（吴祈宗 / 机械工业出版社 / 2013 / textbook）
